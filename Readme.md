
# gogetway

中文 | [English](Readme_EN.md)

## 介绍: 

gogetway是一款开源的能支持TCP流量的高性能流量录制网关，适用于录制TCP流量，支持自定义级别的TCP流量回放，
适用于HTTP1.0-HTTP2.0协议、WebSocket、SSH等基于TCP的协议

## 特性:

1. 可作为库直接进行二次开发，我们提供完整的开发接入文档(文档在做)
2. 也可使用release页面中的安装包即开即用(在做)
3. 占用超低的CPU和内存

## 适用场景:

1. 对于蜜獾服务器可使用其进行原始的流量记录，便于溯源、审计
2. 对于CTF练习题目，可进行流量录制，方便复现
3. 对于线上灰度测试、流量监测，可进行流量录制，方便测试
4. 可捕捉SSH协议等非常规的TCP流量，利于监控审计
5. 可支持服务熔断功能开发，对于异常流量可直接拦截，进行熔断处理
